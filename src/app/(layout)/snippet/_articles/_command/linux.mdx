# Linux 常用命令行

以下默认是 Ubuntu 系统。

```bash
# 编辑 crontab
EDITOR=vim crontab -e

# 生成 SSH 密钥
ssh-keygen -t rsa -b 4096 -C "<名称>"

# 正确设置 SSH 密钥的权限
chmod 700 ~/.ssh
chmod 600 ~/.ssh/*

# 新增用户
sudo adduser <用户名>

# 查看用户名和用户组
sudo vi /etc/passwd

# 创建用户组
sudo groupadd <用户组>

# 将用户加入用户组
sudo gpasswd -a <用户名> <用户组>

# 编辑权限
#  u 文件所有者，g 文件所有者的用户组，o 其他人，a 所有人
#  + 增加权限，- 取消权限，= 设置权限
#  r 读，w 写，x 执行，X 只有当文件为目录文件，或者其他类型的用户有可执行权限时，才将文件权限设置可执行
#  -R 递归设置子目录的权限
sudo chmod o+w -R .
```

更多配置，可以参考 [Bash 备忘清单](https://code.paperplane.cc/docs/bash.html)（自部署）。
