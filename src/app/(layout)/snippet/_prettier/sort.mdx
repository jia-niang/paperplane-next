需要配合 `@trivago/prettier-plugin-sort-imports` 插件（[文档](https://github.com/trivago/prettier-plugin-sort-imports#readme)），它可以为 prettier 提供 `import` 语句分组排序功能。

依赖项：

```bash
npm add -D @trivago/prettier-plugin-sort-imports
```

安装这个包之后，请在 `.prettierrc` 文件中添加以下内容：

```json
{
  "plugins": ["@trivago/prettier-plugin-sort-imports"],
  "importOrder": [
    "^@?[a-zA-Z]+.+",
    "^@/(?!.*\\.(css|scss|less)$)",
    "^\\.\\.?/(?!.*.(css|scss|less)$)",
    "^.(?!(css|scss|less)$)[^.]*$",
    "(.css|.scss|.less)$"
  ],
  "importOrderSeparation": true,
  "importOrderSortSpecifiers": false,
  "importOrderGroupNamespaceSpecifiers": false
}
```

这个配置可以使你的 `import` 按照类型分组：

1. 直接用包名导入的在最上面；
2. 随后是 `@/` 开头的导入；
3. 随后是 `./` 和 `../` 开头的导入；
4. 样式表类文件会放在最后面。

如果是 Next.js 项目，还可以在 `importOrder` 开头加入 `"^server-only$"`，这样使得 `import 'server-only'` 始终排在最顶部。
