给 Prettier 使用的 `.prettierrc` 配置文件，具体配置项可参考 [官方文档](https://prettier.io/docs/en/options)，需要已安装 [prettier](https://prettier.io/) 和 [VSCode prettier 格式化插件](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode)。

以下是我使用的配置，偏好是**单引号**以及**无分号结尾**：

```json
{
  "printWidth": 100,
  "singleQuote": true,
  "semi": false,
  "trailingComma": "es5",
  "tabWidth": 2,
  "arrowParens": "avoid",
  "endOfLine": "lf"
}
```
